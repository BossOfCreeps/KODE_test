## Alembic  
Init:  
``alembic init alembic``  
Create migrations "by hands"  
``alembic revision -m "COMMENT"``  
Autogenerated migrations  
``alembic revision --autogenerate -m "COMMENT"``  
Run migrations  
``alembic upgrade head``
